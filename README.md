# ⛅️ Weather CLI

Консольная утилита для вывода погоды.

---

#### Установка приложения:

Введите в терминале:

```
npm i -g weather-fera
```

Ознакомьтесь со справкой:

```
weather -h
```

#### Перед использованием приложения:

1. Сохраните токен[^1]:
   [^1]: Токен получить в личном кабинете на сайте https://openweathermap.org

```
weather -t [API_KEY]
```

2. Установите город:

```
weather -s [CITY]
```

---

#### Запуск приложения

Введите в терминале:

```
weather
```
